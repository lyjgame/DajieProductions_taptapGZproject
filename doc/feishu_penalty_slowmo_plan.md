# 点球 Slowmo：54321

## 核心玩法

- 游戏类型：2D 双人在线对战 / 足球点球大战 / 鼠标操作小游戏。
- 一句话描述核心机制：玩家分别扮演点球手和守门员，射门瞬间进入 5 秒 slowmo 子弹时间，点球手拖拽足球决定飞行轨迹，守门员拖拽角色决定扑救方向，在 `5、4、3、2、1` 的倒计时压力下完成点球对决。
- 核心主题：`54321` 是每次点球的 5 秒 slowmo 操作窗口，不只是装饰性倒计时。
- 对战形式：1v1 在线联机，一方射门，一方守门，每轮后交换攻守。
- 比赛规则：采用现实点球大战规则，常规 5 球，平局后进入突然死亡。

## 操作方式

- 全程只使用鼠标操作。
- 点球玩家：鼠标点击足球，按住后向后拖动，松开后确认射门。
- 点球玩家输入含义：点击足球的位置决定触球点，拖动方向决定射门角度，拖动距离决定射门力量。
- 守门员玩家：鼠标点击守门员，按住后向左右两侧或目标方向拖动，松开后守门员扑向该处。
- 守门员特殊操作：松开后双击屏幕，守门员会提前移动，扑救范围更大。
- 守门员无输入：默认防守中路，会接住踢向中上的球，用脚踢走踢向中下的球。
- 球门判定区域：左上、左下、中路、右上、右下 5 个区域。

## 关卡设计

- 场景描述：2D 点球手背后伪透视球场，画面下方是点球手和足球，画面上方是球门、守门员、球网和观众席。
- 画面层级：天空和灯光、观众席、中线列队队员、草地球场、球门球网、守门员、足球轨迹、点球手、UI 倒计时。
- 阶段 1 入场：点球手站在点球点后，守门员站在球门中央，裁判在一侧，远处双方队员列队观看。
- 阶段 2 准备主罚：裁判哨响，点球手自动助跑，画面轻微缩放或抖动，心跳声增强。
- 阶段 3 slowmo 操控：球离脚后进入 5 秒慢动作，屏幕显示巨大 `5 -> 4 -> 3 -> 2 -> 1`，双方完成鼠标输入。
- 阶段 4 结果结算：恢复正常速度，播放进球、扑出、接住、脚挡、射偏、门柱或横梁等结果动画，更新比分并进入下一轮。
- 道具：不设计额外道具，优先保证点球对决简单清晰。
- 障碍物：无传统障碍物，主要挑战来自守门员扑救、射门力量、射门角度和倒计时压力。

## 美术风格

- 整体风格：2D Q 版角色 + 伪透视足球场。
- 角色风格：大头短身、动作夸张、表情明显，点球手和守门员用不同球衣颜色区分。
- 场景风格：夜间或黄昏球场，聚光灯照向点球点和球门，观众席用剪影或重复图案表现。
- UI 风格：巨大倒计时数字、强对比比分板、简洁清楚的拖拽箭头和力量提示。
- 参考方向：Q 版体育小游戏、街机点球游戏、2D 伪透视足球场、带慢动作特效的卡通竞技游戏。
- 关键素材：点球手帧动画、守门员帧动画、裁判吹哨、球门球网、足球飞行拖尾、球网震动、倒计时冲击特效、观众欢呼/叹息音效。
- 素材关键词：2D Q 版足球点球手，2D Q 版守门员，点球大战，伪透视球场，点球手背后视角，足球球门，紧张倒计时，5 秒子弹时间，鼠标拖拽射门，简单帧动画。

## 胜负条件

- 怎么赢：常规 5 轮点球结束后，进球数更多的一方获胜。
- 怎么输：常规 5 轮点球结束后，进球数更少的一方失败。
- 平局处理：5 轮后平局进入突然死亡。
- 突然死亡：双方各踢 1 球，如果一方进球而另一方没进，则进球方获胜。
- 单次点球进球：足球最终落点避开守门员扑救区域，并且没有射偏、打门柱弹出或被挡出。
- 单次点球失败：足球被扑出、被接住、被脚挡、射偏、打中门柱或横梁弹出。
