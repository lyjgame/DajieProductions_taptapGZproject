# 界面 04：创建房间界面

## 用途

房主创建好友房，并把房间码分享给另一名玩家。

## UI 内容

- 标题：创建房间
- 房间码显示，例如 `ABCD`
- 按钮：复制房间码
- 状态文字：等待对手加入
- 按钮：返回

## 交互

- 进入界面时自动创建房间并生成房间码。
- 点击“复制房间码”复制房间码。
- 对手加入后跳转到房间等待界面。
- 点击“返回”取消房间并回到模式选择界面。

## 状态

- 创建中：正在创建房间...
- 等待中：等待对手加入...
- 创建失败：网络连接失败
- 对手加入：对手已加入

## 验收标准

- 能生成可用房间码。
- 房间码清晰可见。
- 对手加入后能进入等待/准备流程。
