# 界面 08：入场 / 本轮开始界面

## 用途

每球开始前展示当前轮次、比分和玩家身份，建立点球大战仪式感。

## UI 内容

- 当前轮次：第 X 球
- 当前比分：A 0 : 0 B
- 身份提示：你是点球手 / 你是守门员
- 点球记录：`- - - - -`
- 按钮或自动提示：准备主罚

## 交互

- 可自动停留 1 秒后进入主罚准备界面。
- 时间不够时可以跳过，直接在比赛主界面顶部显示这些信息。

## 状态

- 普通轮次。
- 第 5 球关键轮次。
- 突然死亡轮次。

## 验收标准

- 玩家清楚知道当前身份。
- 玩家清楚知道当前比分和轮次。
- 不影响比赛节奏。
