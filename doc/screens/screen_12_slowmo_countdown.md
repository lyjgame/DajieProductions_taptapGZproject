# 界面 12：Slowmo 倒计时界面

## 用途

展示主题核心：球离脚后的 5 秒 slowmo 输入窗口。

## UI 内容

- 巨大倒计时数字：5、4、3、2、1
- 阶段文字：Slowmo 输入中
- 点球手或守门员对应操作 UI
- 当前比分
- 当前轮次

## 交互

- 倒计时期间允许当前身份玩家完成输入。
- 倒计时结束后锁定输入。
- 未输入时使用默认规则。

## 状态

- 5：进入 slowmo。
- 4：允许输入。
- 3：输入反馈最明显。
- 2：提示即将锁定。
- 1：最后确认。
- 0：进入结算。

## 验收标准

- 倒计时完整显示。
- 数字足够醒目。
- 倒计时结束后输入被锁定。
- `54321` 主题在此界面最清楚。
