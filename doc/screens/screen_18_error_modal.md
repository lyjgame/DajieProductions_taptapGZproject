# 界面 18：异常与提示弹窗

## 用途

处理房间错误、网络等待、对手断线等异常情况。

## UI 内容

- 弹窗标题
- 错误或提示文案
- 主按钮：确认 / 返回
- 可选按钮：重试 / 等待重连

## 常见文案

- 房间不存在
- 房间已满
- 房间码不能为空
- 网络连接失败
- 等待对手输入
- 对手断开连接
- 等待对手重连

## 交互

- 点击“确认”关闭弹窗。
- 点击“返回”回到房间或主界面。
- 点击“重试”重新请求。

## 状态

- 普通错误。
- 网络等待。
- 对手断线。
- 重连中。

## 验收标准

- 所有异常都有提示。
- 玩家知道下一步能做什么。
- 弹窗不会导致游戏流程卡死。
