# 界面 10：点球手视角 / 主罚界面

## 用途

点球玩家在 slowmo 中通过鼠标拖拽足球完成射门输入。

## UI 内容

- 身份标签：你是点球手
- 足球可点击高亮
- 拖拽箭头
- 力量提示：低 / 中 / 高
- 球门 5 区域提示：左上、左下、中路、右上、右下
- 倒计时数字
- 当前比分和轮次

## 交互

- 鼠标点击足球。
- 按住后向后拖动。
- 拖动方向影响射门角度。
- 拖动距离影响射门力量。
- 松开后锁定射门输入。

## 状态

- 未点击足球。
- 正在拖拽。
- 已松开锁定。
- 等待对手输入。

## 验收标准

- 玩家能清楚看到拖拽方向和力量。
- 松开后射门输入被正确记录。
- UI 不遮挡球门和足球轨迹。
