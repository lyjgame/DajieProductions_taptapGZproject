# 界面 11：守门员视角 / 扑救界面

## 用途

守门员玩家在 slowmo 中通过鼠标拖拽守门员完成扑救输入。

## UI 内容

- 身份标签：你是守门员
- 守门员可点击高亮
- 扑救方向箭头
- 提示：拖拽守门员扑救
- 提示：双击提前移动
- 球门 5 区域提示
- 倒计时数字
- 当前比分和轮次

## 交互

- 鼠标点击守门员。
- 按住并拖向目标方向。
- 松开后锁定扑救方向。
- 松开后双击屏幕触发提前移动。
- 不输入时默认守中路。

## 状态

- 未输入：守中路。
- 正在拖拽。
- 已锁定扑救方向。
- 已触发提前移动。
- 等待对手输入。

## 验收标准

- 玩家能清楚看到扑救方向。
- 双击提前移动有反馈。
- 不输入时能按规则守中路。
