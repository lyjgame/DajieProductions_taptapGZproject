# 界面 01：标题 / 主界面

## 用途

玩家进入游戏后的第一个界面，用来展示游戏名称、主题氛围，并引导玩家开始游戏。

## UI 内容

- 游戏标题：点球 Slowmo：54321
- 副标题：5 秒，决定胜负
- 主按钮：开始游戏
- 次按钮：操作说明
- 背景：点球点、球门、巨大半透明 `5 4 3 2 1`

## 交互

- 点击“开始游戏”进入模式选择界面。
- 点击“操作说明”进入操作说明界面。

## 状态

- 默认状态：显示标题和按钮。
- 悬停状态：按钮高亮。
- 加载状态：进入下一界面前可短暂显示 loading。

## 验收标准

- 玩家能看到游戏名和 `54321` 主题。
- “开始游戏”可以进入下一界面。
- UI 不遮挡主要背景视觉。
