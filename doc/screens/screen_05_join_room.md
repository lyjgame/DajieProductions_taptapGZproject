# 界面 05：加入房间界面

## 用途

让第二名玩家输入房间码加入好友房。

## UI 内容

- 标题：加入房间
- 输入框：输入房间码
- 按钮：加入
- 按钮：返回
- 错误提示区域

## 交互

- 输入房间码后点击“加入”。
- 加入成功后进入房间等待界面。
- 点击“返回”回到模式选择界面。

## 状态

- 默认：等待输入房间码。
- 输入为空：提示“房间码不能为空”。
- 房间不存在：提示“房间不存在”。
- 房间已满：提示“房间已满”。
- 网络错误：提示“网络连接失败”。

## 验收标准

- 输入正确房间码可以加入。
- 错误情况有明确提示。
- 不会因为错误房间码卡死。
