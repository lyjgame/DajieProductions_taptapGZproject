# 点球 Slowmo：开发任务拆分

## 目标

把《点球 Slowmo：54321》的 2D 点球大战方案拆成适合 GameJam agent 执行的开发任务。优先保证“可玩闭环”，再补规则、联机和表现。

## 第一阶段：可玩闭环

- 做出 2D 点球手背后伪透视场景。
- 放置点球手、足球、球门、守门员。
- 实现点球手鼠标点击足球并拖拽射门。
- 实现守门员鼠标点击守门员并拖拽扑救。
- 实现球离脚后的 5 秒 slowmo 倒计时。
- 实现 5 区域球门判定：左上、左下、中路、右上、右下。
- 实现基础结果：进球、扑出、射偏。
- 每次点球结束后能重新开始下一次点球。

完成标准：单机环境下能完整踢一球，并看到明确结果。

## 第二阶段：完整点球大战规则

- 实现双方轮流射门和守门。
- 实现每名玩家常规 5 球。
- 实现比分板。
- 实现每轮点球记录。
- 实现 5 轮后的胜负判断。
- 实现平局后的突然死亡。
- 实现最终胜利和失败界面。

完成标准：不考虑联机时，游戏能跑完整场点球大战。

## 第三阶段：双人在线联机

- 实现创建房间。
- 实现输入房间码加入房间。
- 实现双方准备状态。
- 实现玩家身份分配：点球手 / 守门员。
- 同步点球手拖拽输入结果。
- 同步守门员拖拽输入结果。
- 同步每球结算结果。
- 同步比分和轮次。
- 处理玩家断线或等待。

完成标准：两名玩家可以进入同一房间，轮流完成点球大战。

## 第四阶段：基础美术与反馈

- 制作或生成 2D Q 版点球手。
- 制作或生成 2D Q 版守门员。
- 制作球门、球网、足球、草地、观众席背景。
- 添加点球手待机、助跑、射门动画。
- 添加守门员待机、左扑、右扑、中路防守动画。
- 添加巨大 `5 4 3 2 1` 倒计时 UI。
- 添加鼠标拖拽箭头。
- 添加射门力量提示。
- 添加扑救方向提示。
- 添加进球、扑出、射失文本反馈。

完成标准：玩家能清楚看懂自己做了什么、对方做了什么、为什么进球或被扑出。

## 第五阶段：声音与紧张感

- 添加裁判哨声。
- 添加心跳声。
- 添加倒计时滴答声。
- 添加踢球声。
- 添加扑救声。
- 添加球网声。
- 添加观众欢呼和叹息。
- 倒计时越接近 1，音效越紧张。
- 最后一球或关键球加强音效和画面震动。

完成标准：点球前和 slowmo 期间能明显感到紧张。

## 第六阶段：可选打磨

- 添加裁判吹哨动画。
- 添加入场短镜头。
- 添加门将提前移动残影。
- 添加球网震动。
- 添加门柱和横梁结果。
- 添加关键球慢动作回放。
- 添加点球手庆祝和懊恼动画。
- 添加守门员庆祝和失落动画。
- 添加不同球衣颜色选择。
- 添加简单胜场记录或排行榜。

完成标准：游戏展示效果更完整，但这些内容不是 MVP 必需。

## MVP 最小范围

如果时间不够，只保留以下内容：

- 2D 伪透视点球场景。
- 鼠标拖拽射门。
- 鼠标拖拽扑救。
- 5 秒 slowmo 倒计时。
- 5 区域判定。
- 进球 / 扑出 / 射偏。
- 双方轮流。
- 5 球 + 突然死亡。
- 简单房间联机。
- 基础比分板。

## 风险提醒

- 联机同步不要做太复杂，优先同步最终输入和结算结果。
- 美术不要追求复杂动画，优先保证动作读得懂。
- slowmo 不要做真实物理模拟，优先做清晰的 2D 轨迹和区域判定。
- 特效和回放放到最后，先保证点球闭环。
- 如果时间紧，裁判、观众和入场镜头都可以弱化。

## 给 Agent 的执行原则

- 先让一球能玩，再做一整局。
- 先做本地逻辑，再接联机。
- 先做判定，再做动画。
- 先做清楚反馈，再做华丽特效。
- 保留 `54321` slowmo，因为这是主题核心。
