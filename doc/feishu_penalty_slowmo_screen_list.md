# 点球 Slowmo：需要开发的界面清单

## 目标

这份文档先列出《点球 Slowmo：54321》需要开发的所有界面，方便后续按界面继续拆分功能、UI、素材和验收标准。

## 界面总览

建议把游戏界面分成 5 类：

- 启动与主菜单界面
- 联机房间界面
- 比赛过程界面
- 结算界面
- 异常与提示界面

## 1. 启动与主菜单界面

### 1.1 标题 / 主界面

- 用途：玩家进入游戏后的第一个界面。
- 内容：游戏标题、主题视觉、开始游戏按钮。
- MVP：必须做。

### 1.2 模式选择界面

- 用途：选择创建房间或加入房间。
- 内容：创建房间、加入房间、返回按钮。
- MVP：必须做。

### 1.3 操作说明界面

- 用途：简单说明点球手和守门员的鼠标操作。
- 内容：拖拽足球射门、拖拽守门员扑救、双击提前移动。
- MVP：可选。时间不够时可以放到等待界面或比赛提示里。

## 2. 联机房间界面

### 2.1 创建房间界面

- 用途：房主创建好友房。
- 内容：生成房间码、复制房间码、等待对手加入。
- MVP：必须做。

### 2.2 加入房间界面

- 用途：另一名玩家输入房间码加入。
- 内容：房间码输入框、加入按钮、错误提示。
- MVP：必须做。

### 2.3 房间等待界面

- 用途：显示双方是否进入房间。
- 内容：玩家 1、玩家 2、等待状态、返回按钮。
- MVP：必须做。

### 2.4 双方准备界面

- 用途：双方确认开始比赛。
- 内容：准备按钮、双方准备状态、开始倒计时。
- MVP：必须做。

## 3. 比赛过程界面

### 3.1 入场 / 本轮开始界面

- 用途：展示当前轮次、当前谁是点球手、谁是守门员。
- 内容：当前比分、本轮序号、身份提示。
- MVP：建议做。时间不够可简化成文字提示。

### 3.2 主罚准备界面

- 用途：裁判哨响，点球手准备助跑。
- 内容：点球手、足球、守门员、球门、简单提示。
- MVP：必须做，可与比赛主界面共用。

### 3.3 点球手视角 / 主罚界面

- 用途：点球玩家进行射门操作。
- 内容：足球、拖拽箭头、力量提示、目标区域提示、身份标签“你是点球手”。
- MVP：必须做。

### 3.4 守门员视角 / 扑救界面

- 用途：守门员玩家进行扑救操作。
- 内容：守门员、扑救拖拽箭头、提前移动提示、身份标签“你是守门员”。
- MVP：必须做。

### 3.5 Slowmo 倒计时界面

- 用途：球离脚后进入 5 秒 slowmo，是主题核心界面。
- 内容：巨大 `5 4 3 2 1` 倒计时、输入锁定提示、心跳/滴答反馈。
- MVP：必须做。

### 3.6 等待对手输入界面

- 用途：联机时本地已完成输入，但还在等待对手或结算结果。
- 内容：等待对手输入、本轮结算中、加载动画。
- MVP：必须做。

### 3.7 单球结果界面

- 用途：展示这一球进球、扑出或射失。
- 内容：大字结果、简短原因、比分更新。
- MVP：必须做。

## 4. 结算界面

### 4.1 单轮比分更新界面

- 用途：每球结束后展示比分变化和点球记录。
- 内容：当前比分、双方点球记录、下一球提示。
- MVP：建议做。时间不够可合并进单球结果界面。

### 4.2 突然死亡提示界面

- 用途：5 轮后平局时提示进入突然死亡。
- 内容：突然死亡、大号警告、下一球决定胜负。
- MVP：必须做，如果实现突然死亡规则。

### 4.3 比赛胜利界面

- 用途：玩家获胜后的最终结算。
- 内容：你赢了、最终比分、再来一局、返回房间、返回主页。
- MVP：必须做。

### 4.4 比赛失败界面

- 用途：玩家失败后的最终结算。
- 内容：你输了、最终比分、再来一局、返回房间、返回主页。
- MVP：必须做。

## 5. 异常与提示界面

### 5.1 房间错误提示

- 用途：处理房间不存在、房间已满、房间码为空等情况。
- 内容：错误原因、确认按钮。
- MVP：必须做。

### 5.2 网络等待提示

- 用途：处理短时间网络延迟。
- 内容：等待对手、连接中、请稍候。
- MVP：必须做。

### 5.3 对手断线提示

- 用途：处理对手断开连接。
- 内容：对手断开连接、返回房间、返回主页。
- MVP：必须做。

### 5.4 重连提示界面

- 用途：允许玩家短时间等待对手重连。
- 内容：等待对手重连、倒计时、退出按钮。
- MVP：可选。

## MVP 必须开发界面

如果只做最小可提交版本，至少需要：

- 标题 / 主界面
- 模式选择界面
- 创建房间界面
- 加入房间界面
- 房间等待界面
- 双方准备界面
- 主罚准备界面
- 点球手视角 / 主罚界面
- 守门员视角 / 扑救界面
- Slowmo 倒计时界面
- 等待对手输入界面
- 单球结果界面
- 突然死亡提示界面
- 比赛胜利界面
- 比赛失败界面
- 房间错误提示
- 网络等待提示
- 对手断线提示

## 可以合并的界面

为了节省 GameJam 开发时间，可以合并：

- 主罚准备界面、点球手视角、守门员视角、Slowmo 倒计时界面：合并为比赛主界面，根据玩家身份和阶段切换 UI。
- 单球结果界面、单轮比分更新界面：合并为本球结算界面。
- 胜利界面、失败界面：合并为比赛结算界面，根据结果替换标题和颜色。
- 网络等待提示、对手断线提示：可以使用同一个弹窗组件。

## 推荐开发顺序

1. 标题 / 主界面
2. 模式选择界面
3. 创建房间界面
4. 加入房间界面
5. 房间等待界面
6. 双方准备界面
7. 比赛主界面
8. 点球手操作 UI
9. 守门员操作 UI
10. Slowmo 倒计时 UI
11. 单球结果界面
12. 比赛结算界面
13. 错误和断线提示

## 给 Agent 的提醒

- 先做界面流程能跑通，再美化界面。
- 比赛主界面是核心，其他界面都可以简化。
- 点球手视角和守门员视角必须清楚区分。
- Slowmo 倒计时必须醒目，因为这是主题核心。
- 所有联机等待状态都要有提示，避免玩家以为游戏卡住。
